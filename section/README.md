# 剑来世界 章节介绍
剑来目前一共写到第九卷。

## 第一卷 笼中雀
骊珠洞天里有一个小镇以烧窑出名，给皇宫提供了大量的精美瓷器。
每隔一段时间，就会有一批外人进入小镇寻宝。
齐静春已经守护小镇近六十年。
齐静春死后，山崖书院也不复存在，这些学生们需要去大隋书院里继续求学。

## 第二卷 山水郎
